# Lección 5: Proyecto Final + One Line Supremo

## Proyecto Final (`lessons/08_proyecto_final.bas`)

Integra en un solo programa:
- Arrays numéricos y matriz 2x2 con `DIM`
- Carga de datos con `DATA/READ`
- Cálculo complejo con `ROUND`, `SQR`, potencias, `ABS`, `FACT`, `EXP/LN`
- Memoria con `DEFSEG`, `POKE`, `PEEK`
- Parsing de texto con `LEFT$`, `MID$`, `RIGHT$`
- Despacho con `ON ... GOSUB` y `ON ... GOTO`

## One Line Supremo (`lessons/08_one_line_supremo.bas`)

Versión extrema en una sola línea usando `:` para encadenar:
- cálculos complejos
- arrays
- memoria
- strings

## Ejecuta

```bash
node run_prg.js lessons/08_proyecto_final.bas
node run_prg.js lessons/08_one_line_supremo.bas
```

Si quieres, te lo adapto a tu estilo (más rápido, más legible o más "demencial").
